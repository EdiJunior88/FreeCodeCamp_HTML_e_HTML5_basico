Remover comentário no HTML

Comentar é uma forma de deixar comentários para outros desenvolvedores dentro de seu código sem afetar o resultado exibido para o usuário final. Também é uma maneira conveniente de deixar código inativo sem ter de excluí-lo completamente. Comentários em HTML começam com <!-- e terminam em --> Remova seus elementos h1, h2 e p dos comentários.
